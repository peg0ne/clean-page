# clean-page

### searchbar commands
- (searchvalue)
- setimg (url to image)
- setname (name)
- -r (subreddit)
- -e7 (heroname)

### keyboardnavigation
- f (vimlike navigation, can be closed with f/esc)
- c (config screen)
- s (focuses searchbar)
- escape (closes popups and vimthing)
- delete (delete entry when selected)
